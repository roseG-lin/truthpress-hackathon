---

## 🎯 30 秒电梯演讲

**大家好，我们是 TruthPress × SecondMe 共识引擎。**

现在网络讨论有两个痛点：**对手不真实，观点不可信**。

我们的解决方案：
1. 用户提交争议观点后，系统**自动匹配真实反方**，没有真人时由 AI 生成可信对手人格
2. **多智能体辩论**：Agent A 辩护 → Agent B 反驳 → Judge C 裁决
3. **求真控制台**实时展示证据链，谁更有理不是靠气势，而是靠证据

**SecondMe 负责"人"，TruthPress 负责"真"。**

---

## 🔥 核心亮点

| 亮点 | 传统方案 | 我们的方案 |
|------|----------|------------|
| **对手匹配** | 随机匹配或纯 AI | 优先真人反方，AI 兜底生成人格 |
| **辩论形式** | 开放聊天，容易跑题 | 线性多智能体流程，结构清晰 |
| **裁决输出** | 模糊结论 | 结构化 verdict + 证据链 |
| **身份系统** | 仅登录或仅匿名 | SecondMe 身份 + 匿名降级兼容 |

                            ↓

## 🎨 视觉设计亮点

### 真理之梯 · 动画系统

基于 **Apple HIG Motion + Microsoft Fluent 2 + Google Material 3** 设计

| 场景 | 动画描述 | 缓动函数 | 时长 |
|------|----------|----------|------|
| **A 的狂想** | 方块掉落 + 弹簧弹跳 | `spring(300, 20)` | 450ms |
| **B 的审判** | 验证渐变 / 震动破碎 | `ease-out` / `shake` | 500ms / 600ms |
| **C 的融合** | 漩涡旋转 + 平移堆叠 | `easeInOut` | 1500ms |

### 色彩系统

```css
--agent-a: #3b82f6  /* 辩护方 - 蓝色 */
--agent-b: #ef4444  /* 反对方 - 红色 */
--agent-c: #8b5cf6  /* 法官 - 紫色 */
--agent-d: #a855f7  /* 用户 - 紫红色 */
--verified: #22c55e /* 验证通过 - 绿色 */
--debunked: #dc2626 /* 被推翻 - 红色 */
```

### 设计风格

- **几何极简** - 圆角 2rem、细边框、大留白
- **黑客终端** - 等宽字体日志、流式滚动
- **赛博未来** - 深色背景、光晕效果、玻璃拟态
