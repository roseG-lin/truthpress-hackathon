## 📝 如果需要分离组件

如果团队成员希望将动画组件分离到独立文件，可以这样组织：

### 建议的文件结构

```
src/
├── app/
│   └── page.tsx                      # 主页面（简化版）
├── components/
│   └── home/
│       ├── TruthLadderLogo.tsx       # Logo 组件
│       ├── AgentCard.tsx             # Agent 卡片
│       ├── NavBar.tsx                # 导航栏
│       └── index.ts                  # 统一导出
└── lib/
    └── home-animations.ts            # 动画变体定义
```

### 示例：分离 TruthLadderLogo

**创建文件：** `src/components/home/TruthLadderLogo.tsx`

```tsx
"use client";

import { motion } from "framer-motion";

const logoVortex = { ... };

const glowPulse = { ... };

  return (
    <div className="relative w-28 h-28 mx-auto mb-6">
      {/* ... 组件内容 ... */}
    </div>
  );
}
```

**然后在 page.tsx 中导入：**

```tsx
import TruthLadderLogo from "@/components/home/TruthLadderLogo";
```
