## 一、趋势调研摘要

### 1.1 大厂案例与核心亮点

| 案例 | 来源 | 核心亮点 | 本项目借鉴点 |
|------|------|----------|--------------|
| **Apple VisionOS 空间交互** | Apple HIG Motion 2025 | 物理弹簧动画、视差层级、材质形变 | 方块掉落的 spring 参数 (stiffness:300, damping:20)、Avatar 悬浮节奏 (1.5s 循环) |
| **Microsoft Fluent 2 Mica 材质** | Fluent 2 Design | 半透明渐变、光效扫描、状态转换 | Terminal 面板的 `backdrop-blur` + 边框光效 (`border-white/10`) |
| **Google Material 3 强调缓动** | Material Design 3 | Emphasized Decelerate 曲线、容器转换 | A 方块掉落的 `cubic-bezier(0.05, 0.7, 0.1, 1)` |
| **字节跳动豆包 AI 思考可视化** | 豆包 Web 2025 | 流式输出脉冲、状态机进度 | SSE 日志的流式滚动 + Agent 状态指示 |

### 1.2 设计原则映射

```
Apple HIG Motion → 物理动画 (弹簧、惯性)
Fluent 2 → 材质系统 (玻璃拟态、光晕)
Material 3 → 缓动函数 (标准、强调、减速)
```

## 三、动画方案

### 3.1 Scene 1: A 的狂想 (The Build)

| 步骤 | 动画描述 | 时长 | 缓动函数 | 术语 |
|------|----------|------|----------|------|
| 方块掉落 | `y: -100 → 0` + 弹跳 | 450ms | `cubic-bezier(0.34,1.56,0.64,1)` | Spring Physics |
| A 跳跃就位 | `layoutId` 共享元素过渡 | 300ms | `ease-out` | 共享元素动画 |
| 等待脉冲 | `scale: [1, 1.05, 1]` | 1000ms 循环 | `ease-in-out` | 关键帧动画 |

### 3.2 Scene 2: B 的审判 (The Check)

#### 验证通过

| 步骤 | 动画描述 | 时长 | 缓动函数 | 术语 |
|------|----------|------|----------|------|
| 颜色渐变 | `#3b82f6 → #22c55e` | 500ms | `ease-out` | Morphing |
| B 移动就位 | `layout` 自动重排 | 300ms | `spring(280, 18)` | 布局动画 |
| ✓ 图标弹出 | `scale: 0 → 1` + 旋转 | 200ms | `spring(400, 15)` | 弹簧动画 |

#### 被推翻

| 步骤 | 动画描述 | 时长 | 缓动函数 | 术语 |
|------|----------|------|----------|------|
| 震动 | `x: [0,-8,8,-8,8,0]` | 600ms | `ease-in-out` | 震动动画 (Shake) |
| 破碎消失 | `scale: 1 → 0` + `opacity: 1 → 0` | 600ms | `ease-in` | 淡出动画 |
| A 向下掉落 | `layout` 自动重排 | 500ms | `spring(300, 20)` | 布局动画 |

### 3.3 Scene 3: C 吸收 D (The Merge)

| 步骤 | 动画描述 | 时长 | 缓动函数 | 术语 |
|------|----------|------|----------|------|
| C 漩涡效果 | `rotate: 0 → 360` + `scale: [1,1.2,1]` | 1000ms | `ease-in-out` | 旋转动画 |
| D 平移 | `x: 300 → 0` + `y: [0,-20,0]` | 1500ms | `cubic-bezier(0.4,0,0.2,1)` | 视差滚动 |
| 堆叠 | `staggerChildren: 100ms` | 500ms | `ease-out` | 交错动画 |
| C 颜色渐变 | `#9ca3af → #8b5cf6` | 2000ms | `linear` | 属性动画 |
| 旗帜举起 | `y: [0,-40]` + `opacity: 0 → 1` | 500ms | `ease-out` | 关键帧动画 |

## 四、前端实现规范

### 4.1 技术栈

```yaml
框架：Next.js 14 (App Router)
语言：TypeScript 5
动画库：Framer Motion 11
样式：Tailwind CSS 3
图标：Lucide React
状态管理：useReducer (React 内置)
```

### 4.2 核心术语标注

| 术语 | 英文 | 应用场景 |
|------|------|----------|
| **缓动函数** | Easing Function | `cubic-bezier` 自定义曲线 |
| **关键帧** | Keyframe | `y: [0, -6, 0]` 数组语法 |
| **属性动画** | Property Animation | `backgroundColor`、`scale` 直接过渡 |
| **Morphing 变形** | Morphing | `layoutId` 共享元素 + `layout` 自动重排 |
| **视差滚动** | Parallax Scrolling | 多层级 `translateY` 差异化 |
| **状态机动画** | State Machine Animation | 根据 `LadderState` 触发不同 `variants` |
| **玻璃拟态** | Glassmorphism | `backdrop-blur` + 半透明边框 |

### 4.3 性能优化

```typescript
// 1. LayoutGroup + AnimatePresence 组合
<LayoutGroup>
  <AnimatePresence mode="popLayout">
    {items.map(item => (
      <motion.div layoutId={item.id} key={item.id} />
    ))}
  </AnimatePresence>
</LayoutGroup>

// 2. 弹簧物理参数优化
transition: {
  type: "spring",
  stiffness: 300,  // 刚度
  damping: 20,     // 阻尼
  mass: 1,         // 质量
}

// 3. 限制方块数量
const MAX_BLOCKS = 20;

// 4. 尊重用户偏好
<MotionConfig reducedMotion="user">
```

## 五、可复用模板

### 5.1 方块掉落动画模板

```typescript
import { motion, Variants } from "framer-motion";

const blockDropVariants: Variants = {
  initial: { y: -120, opacity: 0, scale: 0.85 },
  animate: {
    y: 0,
    opacity: 1,
    scale: 1,
    transition: {
      type: "spring",
      stiffness: 300,
      damping: 20,
      duration: 0.45,
    },
  },
};

export function DropBlock({ text }: { text: string }) {
  return (
    <motion.div
      variants={blockDropVariants}
      initial="initial"
      animate="animate"
      className="rounded-xl bg-blue-500 px-4 py-2 text-white"
    >
      {text}
    </motion.div>
  );
}
```

### 5.2 验证通过颜色渐变模板

```typescript
const verifiedVariants: Variants = {
  pending: { backgroundColor: "#3b82f6" },
  verified: {
    backgroundColor: ["#3b82f6", "#22c55e"],
    scale: [1, 1.05, 1],
    transition: { duration: 0.5, ease: "ease-out" },
  },
};

export function VerifiedBlock({ text }: { text: string }) {
  const [isVerified, setIsVerified] = useState(false);

  return (
    <motion.div
      variants={verifiedVariants}
      animate={isVerified ? "verified" : "pending"}
      onClick={() => setIsVerified(true)}
      className="rounded-xl px-4 py-2 text-white"
    >
      {text}
    </motion.div>
  );
}
```

### 5.4 震动消失动画模板

```typescript
const shakeExitVariants: Variants = {
  exit: {
    x: [0, -8, 8, -8, 8, 0],
    opacity: [1, 1, 1, 0],
    scale: [1, 1, 0.8, 0],
    transition: { duration: 0.6, ease: "ease-in-out" },
  },
};

export function ShakeBlock({ text, onExit }: { text: string; onExit: () => void }) {
  return (
    <AnimatePresence>
      <motion.div
        key="block"
        variants={shakeExitVariants}
        initial={{ opacity: 1 }}
        animate={{ opacity: 1 }}
        exit="exit"
        onAnimationComplete={onExit}
        className="rounded-xl bg-red-500 px-4 py-2 text-white"
      >
        {text}
      </motion.div>
    </AnimatePresence>
  );
}
```

## 六、术语表

### UI 前端术语

| 术语 | 英文 | 解释 |
|------|------|------|
| **响应式断点** | Responsive Breakpoints | 不同屏幕尺寸的布局切换点 (sm/md/lg/xl/2xl) |
| **栅格系统** | Grid System | 12 列或自定义列数的布局网格 |
| **玻璃拟态** | Glassmorphism | 半透明 + 模糊 + 边框的视觉效果 |
| **新拟态** | Neumorphism | 阴影塑造的凸起/凹陷效果 |
| **渐进式增强** | Progressive Enhancement | 基础功能优先，高级功能按需加载 |
| **优雅降级** | Graceful Degradation | 高级功能不可用时回退到基础版本 |
| **CSS Transition** | CSS 过渡 | 属性变化的平滑过渡效果 |
| **CSS Animation** | CSS 动画 | 关键帧定义的动画序列 |
| **WebGL** | WebGL | 浏览器中的 3D 图形渲染 API |
| **SVG Sprite** | SVG 雪碧图 | 多个 SVG 图标合并为单个文件 |
| **事件委托** | Event Delegation | 在父元素上监听子元素事件 |
| **视差滚动** | Parallax Scrolling | 不同层级以不同速度滚动 |
| **动态排版** | Dynamic Typography | 根据内容自适应的字体大小/间距 |
| **Mica 材质** | Mica Material | Microsoft Fluent 2 的半透明材质系统 |

### 动画专业术语

| 术语 | 英文 | 解释 |
|------|------|------|
| **缓动函数** | Easing Function | 定义动画速度变化的曲线 (ease-in/out/in-out) |
| **关键帧** | Keyframe | 动画序列中的关键状态点 |
| **帧率** | FPS (Frames Per Second) | 每秒显示的帧数，目标 60fps |
| **变形动画** | Morphing | 形状/颜色的平滑过渡变换 |
| **交互动效** | Interactive Motion | 用户触发或响应的动画 |
| **跟随动作** | Follow Through | 主体停止后附属部分继续运动 |
| **重叠动作** | Overlapping Action | 不同部分以不同节奏运动 |
| **粒子系统** | Particle System | 大量小元素组成的动态效果 |
| **Lottie 动画** | Lottie | Airbnb 开源的 JSON 格式动画文件 |
| **状态机动画** | State Machine Animation | 基于状态转换触发动画 |
| **动作捕捉** | Motion Capture | 记录真实运动数据用于动画 |
| **生物力学建模** | Biomechanical Modeling | 模拟真实生物运动规律 |
| **叙事性动画** | Narrative Animation | 讲述故事或传达情感的动画 |
