## 三、封面图元素详解

### 1. 背景层 (Background Layer)

```css
/* 径向渐变 - 顶部光源 */
background: radial-gradient(
  circle at top,
  #1c2a41 0%,
  transparent 55%
);

/* 线性渐变 - 左右分割 */
left: linear-gradient(
  120deg,
  rgba(8,12,22,0.95),
  rgba(8,12,22,0.75)
);

right: radial-gradient(
  circle at top,
  #0f172a,
  transparent 65%
);
```

## 四、动态封面版本 (GIF/Video)

### 动画序列 (3 秒循环)

| 时间 | 动画内容 | 缓动函数 |
|------|----------|----------|
| 0-0.5s | 方块从顶部落下 | spring(300, 20) |
| 0.5-1.0s | A 跳跃到方块上 | ease-out |
| 1.0-1.5s | B 扫描方块 | ease-in-out |
| 1.5-2.0s | 方块变绿/变红破碎 | ease-out / shake |
| 2.0-2.5s | C 漩涡旋转 | easeInOut |
| 2.5-3.0s | 整体脉冲光晕 | ease-in-out |

### CSS 关键帧动画

```css
@keyframes ladder-build {
  0% {
    transform: translateY(-100px);
    opacity: 0;
  }
  50% {
    transform: translateY(0);
    opacity: 1;
  }
  100% {
    transform: translateY(-6px);
  }
}

@keyframes glow-pulse {
  0%, 100% {
    box-shadow: 0 0 40px rgba(14,165,233,0.15);
  }
  50% {
    box-shadow: 0 0 80px rgba(14,165,233,0.3);
  }
}

@keyframes vortex-spin {
  from {
    transform: rotate(0deg) scale(1);
  }
  50% {
    transform: rotate(180deg) scale(1.2);
  }
  to {
    transform: rotate(360deg) scale(1);
  }
}
```
