## 打印说明

### 打印设置

| 设置项 | 值 |
|--------|-----|
| 纸张尺寸 | A5 (148mm × 210mm) |
| 打印方式 | 双面打印（短边翻转） |
| 纸张类型 | 铜版纸/哑光纸 157g |
| 颜色模式 | CMYK |
| 出血 | 上下左右各 3mm |

### 裁剪尺寸

- **成品尺寸**: 148mm × 210mm
- **含出血**: 154mm × 216mm
- **安全边距**: 距边缘 5mm

### 数量建议

| 用途 | 数量 |
|------|------|
| 评委每人一份 | 5-10 份 |
| 展示台自取 | 20-30 份 |
| 备用 | 10 份 |
| **总计** | **35-50 份** |

## 电子版使用

### 在平板上展示

```
1. 将 PDF 导入 iPad/安卓平板
2. 使用 GoodNotes/Notability 打开
3. 演示时快速翻阅给评委看
```

### 在手机上查看

```
1. 保存为手机壁纸（分两张）
2. 或添加到钱包/备忘录
3. 演示时快速查阅
```
