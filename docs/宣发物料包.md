### 评委问答速查卡

```
Q: 和传统辩论平台有什么区别？
A: 三个核心差异：
   1. 对手匹配：优先真人反方，不是随机匹配
   2. 结构化流程：线性多智能体，不跑题
   3. 事实核查：Judge C 强制输出 verdict + 证据链

Q: 如何保证 AI 对手人格可信？
A: Prompt Engineering 约束：
   - 具体人设（名字、背景、立场）
   - 立场与用户观点概念上相反
   - 论证方式像真实网友

Q: 技术难点？
A: 1. 多智能体编排（协调发言顺序）
   2. SSE 流式动画（实时推送 + 触发动画）

Q: 商业模式？
A: 教育领域（辩论教学）、企业决策（意见汇总）、
   内容审核（事实核查）
```
