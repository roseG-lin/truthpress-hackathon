## 🛠️ 技术实现

### Framer Motion 变体

```typescript
// Logo 漩涡
const logoVortex = {
  animate: {
    rotate: 360,
    scale: [1, 1.05, 1],
    transition: {
      duration: 20,
      repeat: Infinity,
      ease: "linear",
    },
  },
};

// 粒子悬浮
const particleFloat = {
  animate: {
    y: [0, -20, 0],
    opacity: [0.3, 0.8, 0.3],
    transition: {
      duration: 3,
      repeat: Infinity,
      ease: "easeInOut",
    },
  },
};

// 卡片悬停
const cardHover = {
  hover: {
    y: -8,
    scale: 1.02,
    transition: { duration: 0.3, ease: "easeOut" },
  },
};
```

### CSS 渐变

```css
/* 锥形渐变 - 光晕环 */
background: conic-gradient(
  from 0deg,
  #3b82f6,
  #8b5cf6,
  #a855f7,
  #3b82f6
);

/* 线性渐变 - 方块 */
background: linear-gradient(
  to right,
  #3b82f6,
  #60a5fa
);

/* 径向渐变 - 背景 */
background: radial-gradient(
  circle at top,
  #e0f2fe,
  white 35%,
  #f8fafc
);
```

## 📊 性能优化

### 动画性能

| 优化项 | 说明 |
|--------|------|
| **will-change** | Framer Motion 自动处理 |
| **GPU 加速** | 使用 `transform` 和 `opacity` |
| **减少重排** | 避免使用 `top/left` 改用 `translate` |
| **延迟加载** | 组件依次入场避免同时渲染 |

### 加载状态

```typescript
// 骨架屏加载
if (loading) {
  return (
    <main>
      <motion.div
        animate={{ scale: [1, 1.2, 1], rotate: [0, 180, 360] }}
        className="w-16 h-16 rounded-full border-4 border-sky-200 border-t-sky-600 animate-spin"
      />
    </main>
  );
}
```
