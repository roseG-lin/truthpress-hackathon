## 📧 评委问答准备

### Q1: 和传统辩论平台有什么区别？

**A:** 三个核心差异：
1. **对手匹配**：我们优先匹配真实历史反方，不是随机匹配
2. **结构化流程**：线性多智能体，不是开放聊天容易跑题
3. **事实核查**：Judge C 强制输出结构化 verdict + 证据链

### Q2: 如何保证 AI 生成的对手人格可信？

**A:** 我们通过 Prompt Engineering 约束：
- 要求生成具体的人设（名字、背景、立场）
- 要求立场与用户观点概念上相反
- 要求论证方式像真实网友（有情绪、有逻辑漏洞）

### Q3: 技术难点是什么？

**A:** 两个主要难点：
1. **多智能体编排**：需要协调 Agent A/B/C 的发言顺序和内容关联
2. **SSE 流式动画**：辩论过程需要实时推送到前端，同时触发动画

### Q4: 商业模式是什么？

**A:** 潜在方向：
- 教育领域：辩论教学、批判性思维训练
- 企业决策：多方意见结构化汇总
- 内容审核：争议话题事实核查
